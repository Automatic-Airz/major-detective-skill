# Demo Cases

Use these as quick demonstrations for testing the student/family workflow. Creator-facing content demos are optional and secondary.

## Case 1: Biology Anxiety

Input:

```text
女生，山东，物化生，家里建议报生物科学。我喜欢实验，但不想读博，希望以后就业稳定。这个专业靠谱吗？
```

Expected mode: `report` plus parent-facing summary.

## Case 2: Civil-Service-Oriented Comparison

Input:

```text
法学、汉语言文学、新闻传播，我想以后考公，也想保留就业选择，哪个更适合？
```

Expected mode: `compare`.

## Case 3: Psychology Misconception

Input:

```text
我喜欢心理学，觉得可以帮助别人，但家里担心就业。心理学到底是不是坑？
```

Expected mode: `quick` or `report`.

## Case 4: Student Action Checklist

Input:

```text
基于生物科学这个案例，帮我生成一份下一步核验清单，告诉我和家长接下来应该查什么、问什么、比较什么。
```

Expected mode: `action`.

## Optional Case 5: Creator-Facing Script

Use only for投稿 or运营展示.

Input:

```text
帮我把「AI专业侦探分析生物科学」做成一条抖音图文脚本，突出常见误解和风险雷达。
```

Expected mode: `content`.
