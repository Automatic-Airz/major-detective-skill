# Full App Dialogue Flow Example

This example shows how the full experience should feel when the Skill is used as an Agent dialogue, with the offline demo acting as the visual shell.

## Step 1: User Opens The Demo

Screen state:

- Left side: student situation input
- Mode: 专业调查
- Major: 生物科学
- Right side: waiting to generate report

## Step 2: User Enters The First Question

User:

```text
女生，山东，物化生，家里建议报生物科学。我喜欢实验，但不想读博，希望以后就业稳定。这个专业靠谱吗？
```

Agent:

```text
我先把你的真实问题复述一下：你不是单纯想知道「生物科学好不好」，而是在问：

在山东、物化生、家里偏向生物科学、你喜欢实验但不想读博、又希望就业稳定的情况下，这个专业和你的现实目标是否匹配。

当前信息完整度约 60%。我可以先给你基础版调查；如果要更准，后面建议补充目标位次、目标学校层次、是否接受读研、是否排斥化学课程。
```

## Step 3: Agent Generates A First Report

Agent:

```text
## 生物科学一句话画像

生物科学更像生命科学的基础研究训练专业，不是医学，也不是简单做实验。它适合能接受实验、文献阅读和读研路径的人；如果目标是本科毕业就获得稳定就业，需要谨慎核验学校平台和相邻转向。

## 这个专业到底学什么

- 普通生物学、动物学、植物学
- 无机化学、有机化学、分析化学
- 生物化学、细胞生物学、遗传学、分子生物学
- 微生物学、生态学、实验课、文献阅读和毕业论文

## 常见误解

- 生物科学不是医学，通常不能直接成为医生。
- 不是只做有趣实验，也包含化学、理论、文献和重复操作。
- 本科就业通常不如名字听起来直接，需要提前规划方向。

## 适合你的地方

- 你喜欢实验，这和专业训练有一定匹配。
- 你是物化生背景，进入相关课程不会完全陌生。

## 需要谨慎的地方

- 你不想读博，而生物科学的高质量发展路径往往更依赖深造。
- 你希望稳定就业，但生物科学本科就业确定性通常需要看学校平台、城市产业和个人转向能力。
- 如果你排斥化学或重复实验，这个专业体验可能会打折。

## 风险雷达

- 本科就业压力：高
- 读研依赖度：高
- 实验压力：高
- 化学课程压力：中高
- 转行友好度：中
- 信息误解风险：高

## 替代专业

可以同时比较：

- 药学
- 医学检验技术
- 生物医学工程
- 生物信息学
- 食品科学与工程

## 结论

如果你能接受读研，且不排斥化学和长期实验训练，生物科学可以继续调查。

如果你的底线是本科毕业后尽量稳定就业，又明确不想长期深造，它不一定是最稳的第一选择，建议和药学、医学检验技术、生物医学工程、生物信息学一起比较。
```

## Step 4: Agent Asks Focused Follow-Up Questions

Agent:

```text
为了把报告从基础版升级成个人适配版，我建议你只补 4 个信息：

1. 你能接受读研吗？不能接受的是读博，还是连读研也不想？
2. 你是否排斥化学课和长期实验？
3. 你目标大概是省内普通本科、省内重点、211，还是更高层次？
4. 家里说的「稳定」更偏体制内、医院/检测机构、国企，还是只要就业确定就可以？
```

## Step 5: User Adds Context

User:

```text
读研可以接受，但不想读博。化学一般，不算喜欢。目标可能是省内重点或普通一本。家里说的稳定主要是别太难就业，最好能在山东。
```

## Step 6: Agent Updates The Recommendation Logic

Agent:

```text
那判断会更清楚一些：

你不是完全不适合生物方向，但「生物科学」本身不是就业路径最直接的专业。对你来说，更适合的策略不是只看生物科学，而是把它放进一个专业组里比较：

第一梯队继续调查：
- 药学：和生命科学相关，就业场景更靠近药企、注册、生产、药事服务。
- 医学检验技术：路径更具体，但要核验资格、医院招聘和学校资源。
- 生物医学工程：更偏工程交叉，如果你能接受一定数学和设备方向，可以调查。

第二梯队谨慎调查：
- 生物科学：适合接受读研和科研训练的人，但本科就业确定性要核验。
- 生物信息学：前景有交叉性，但要求代码和统计，如果你排斥代码要谨慎。

你接下来最该查的不是网上评价，而是目标学校的培养方案和毕业生去向。
```

## Step 7: User Switches To Parent Version

User:

```text
帮我生成一版给家长看的。
```

Agent:

```text
给家长看的冷静版：

生物科学不是不能选，但不建议只因为「生物有前景」或「孩子选了物化生」就直接确定。这个专业通常更依赖学校平台和后续深造，本科就业路径没有医学检验、药学等专业那么直接。

现在更稳妥的做法是一起核验三件事：

1. 目标学校培养方案：化学、实验、科研训练比例有多高。
2. 毕业去向：本科就业、考研、保研、出国分别是什么比例。
3. 兜底方案：如果孩子大二发现不适合，是否有转专业、辅修、考研转向药学/医学技术/生物信息的空间。

建议不是马上否定生物科学，而是把生物科学、药学、医学检验技术、生物医学工程放在一起比较，再根据孩子能否接受读研、化学实验和山东本地就业机会做决定。
```

## Step 8: User Switches To Action Checklist

User:

```text
帮我生成一份下一步核验清单。
```

Agent:

```text
下一步核验清单：

1. 查官方信息
- 山东省考试院当年招生计划和选科要求
- 目标学校生物科学培养方案
- 学院公开的毕业去向、升学比例、就业质量报告
- 转专业、辅修、保研和实习政策

2. 同步比较相邻专业
- 药学
- 医学检验技术
- 生物医学工程
- 生物信息学
- 食品科学与工程

3. 问家里的问题
- 家里说的稳定具体是体制内、医院/学校、国企，还是只要就业确定？
- 是否接受读研带来的时间成本？
- 如果发现不适合，是否支持转专业、辅修或考研转向？

4. 问自己的问题
- 我能不能接受化学、重复实验、英文文献和科研训练？
- 我喜欢的是实验本身，还是生命科学问题？
- 如果本科就业不够直接，我是否愿意提前做实习和转向规划？

5. 会改变判断的证据
- 目标学校生物科学升学和就业去向非常透明且匹配你的目标。
- 相邻专业的培养方案更接近稳定就业。
- 你发现自己不能接受核心课程或长期深造成本。
```
